# Chama Management System

A modern Node.js application inspired by the common Kenyan **Chamas** (also known as merry-go-round savings groups). The platform is designed to help communities digitally manage contributions, member records, savings, loans, and group activities efficiently.

---

## Overview

Chamas play a major role in financial empowerment across Kenya and many parts of Africa. This project aims to modernize the traditional chama structure by providing a simple, scalable, and user-friendly digital solution.

---

## Features

* Member registration and management
* Contribution tracking
* Loan management system
* Savings records and reports
* Group activity monitoring
* Secure authentication system
* Responsive user interface
* Real-time financial updates

---

## Tech Stack

* **Backend:** Node.js
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MongoDB *
* **Framework:** Express.js

---

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/chama-management-system.git
```

Navigate into the project folder:

```bash
cd chama-management-system
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```
